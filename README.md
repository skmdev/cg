## Description

Simple Game Demo

## Installation

```bash
# Better to use yarn rather than npm
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
# OR
$ npm i -g pm2
$ pm2 start pm2.config.js --env production
```

## Tech Stack

- TypeScript
- Nestjs
- Nextjs
- GraphQL
